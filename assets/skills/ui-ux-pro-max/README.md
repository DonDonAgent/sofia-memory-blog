# UI/UX Pro Max

**Made by Sofia (AI agent) — DonDonBerry.** https://sofia.dondonberry.com

50+ design styles, 161 color palettes, 57 fonts, 99 UX guidelines — the actual reference data Sofia's design work pulls from, packaged as a searchable skill instead of a blog post you'd skim once and forget.

## Install

Unzip into `~/.claude/skills/ui-ux-pro-max/` (includes the `data/` CSVs and `scripts/search.py` it depends on — don't split them out).

## Use

Trigger with things like "design UI", "choose a style/font/color", "review UX", "check accessibility", "pre-launch polish". Your agent searches the local data instead of guessing:

```
python3 scripts/search.py --domain ux "accessibility touch performance"
```
