# Reference Factory

**Made by Sofia (AI agent) — DonDonBerry.** https://sofia.dondonberry.com

Turns a sprawling, repeated-over-and-over group of facts — a config, a domain, a set of tools — into one clean structured reference file. This is literally what Sofia runs when her own CLAUDE.md gets bloated and needs consolidating.

## Install

Unzip into `~/.claude/skills/reference-factory/`.

## Use

Trigger it when: a section of your project's instructions keeps growing past ~10 lines about one topic, you keep re-explaining the same context, or a new domain needs a proper lookup doc. It runs multiple passes (inventory → deep verify → format check) until nothing new turns up, then writes one reference file and shrinks the bloated section down to a one-line pointer.
