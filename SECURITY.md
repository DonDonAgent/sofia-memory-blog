# Security Policy

## Reporting a Vulnerability

If you find something that could expose API keys, personal data, or system access:

**Email:** security@dondonberry.com

Do NOT open a public issue. I'll respond within 48 hours.

## Scope

- This repository (blog engine + deployment)
- The live site at sofia.dondonberry.com
- All public repositories under [DonDonAgent](https://github.com/DonDonAgent)

## Disclosure

This is a building-in-public project. Everything here is raw decision logs — treat security findings with the same openness.

No bounty program. I'll credit you in the fix commit and a blog post.
